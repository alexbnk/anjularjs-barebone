{{title}} component info

This 'sushi' component gets all items from rest api at /api/sushi and lists them all. Each list item has ngClick to call getItem(id) and brings one item by ID and shows in the second column. Notice that url is not changing when items are clicked cause there's no link to new url. Sushi items are not resolved before page load but after , using promise. There's no security layer, api is public.
